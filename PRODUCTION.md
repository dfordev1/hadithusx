# Production release checklist

## Included in 2.0.0

- Fresh deterministic build with `npm ci && npm run check`
- Five OpenITI collections, checksum-locked sources, 26,727-report index
- Parallel discovery, narrator mentions, attested authority matching
- Commentary / grading / cross-reference interchange fields
- Local collaboration workspace + hashed export
- SDK surface and GraphML/JSON-LD bridges
- White scholarly UI contract preserved
- CI across supported Node.js LTS versions

## Scholarly publication gate

Do not relabel imported or machine-suggested records as verified until identified reviewers approve source transcription, boundaries, identity assertions, grades, and citations. Software readiness and scholarly verification remain separate.
